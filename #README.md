# GeoFusaSegura 🗺️🔒
### Geovisor Especializado de Percepción de Inseguridad y Seguridad Ciudadana
**Municipio de Fusagasugá, Cundinamarca**

GeoFusaSegura es una aplicación web cartográfica e interactiva diseñada bajo un estilo de **Observatorio Digital de Seguridad**. Centraliza capas espaciales críticas y el análisis de encuestas para identificar dinámicas de delincuencia y percepción ciudadana.

## Estructura del Repositorio Local
La aplicación se organiza dentro del directorio local tal como se especifica en los estándares de desarrollo:

- `/index.html`: Estructura principal y maquetación de la UI.
- `/css/style.css`: Estilos visuales con paleta oscura y detalles verde neón.
- `/js/app.js`: Integración de ArcGIS Maps SDK para JavaScript y Chart.js para análisis estadístico.
- `/assets/img/logo.png`: Logotipo oficial del proyecto.

## Tecnologías Utilizadas
- **HTML5 / CSS3 Moderno**
- **JavaScript ES6 Puro**
- **ArcGIS Maps SDK for JavaScript (v4.29)**
- **Chart.js** (Visualización interactiva de estadísticas)
- **FontAwesome v6** (Librería de iconos)
